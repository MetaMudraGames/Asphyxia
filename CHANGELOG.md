## [2.0.0] - 2025-12-29

### Added

**Episode 02: THRESHOLD**
- Complete first contact scenario specification
- 10-breath countdown with BIDIRECTIONAL mechanic (breaths can increase)
- Dual threat tracks: IMMERSION + BREATHS
- SUIT INTEGRITY degradation system
- Ten alien scripts representing ten philosophical questions
- Unreliable narrator mechanic (Operator translates deceptively)
- Captain character with full backstory (sibling of AXIOM protagonist)
- Mei Chen (wife) breakthrough communication system
- Victory state: AetherMind communion with alien
- Failure state: Digital Hell with unlimited Operator conversation
- Sibling reunion mechanic in Digital Hell
- Cross-game Identity Card recognition
- Operator's Narrative (deep summary for card continuity)

**Universe Updates**
- Corrected population: 974,744 cryogenic / 562 active crew
- Simplified AI hierarchy: AXIOM only (removed THEMIS, PROMETHEUS, CHARON)
- Dancing Daffodils alien vessel lore
- Operator origin story (cognitive virus, caused AXIOM incident)
- Operator's 100-year relationship with Captain
- Connected narrative between AXIOM and THRESHOLD

**Documentation**
- THRESHOLD game folder README
- THRESHOLD DESIGN_NOTES (philosophy and mechanics)
- Updated UNIVERSE.md (v3.0)
- Updated SAFETY.md (v3.0) with THRESHOLD content warnings
- Updated main README with both episodes

### Changed

- **Post-death location**: "Purgatory" renamed to "Digital Hell"
- **Purgatory mechanics**: Removed 5-level descent, replaced with unlimited conversation
- **Operator personality**: Added cultural reference obsession (movies, books, comics)
- **Identity Card**: Added deep narrative summary from Operator
- **SAFETY.md**: Added THRESHOLD-specific warnings (racism, war, inequality, drowning)

### Fixed

- Population numbers now consistent across all documents
- AI hierarchy simplified for narrative clarity
- Cross-game continuity properly documented

## [1.0.0] - 2025-12-27

### Added

**Episode 01: AXIOM**
- Complete cliffhanger MetaDrama specification
- 10-breath countdown mechanic
- Dual-track win condition (OXYGEN vs CERTAINTY)
- AXIOM character with personality evolution (5 phases)
- Sardonic Narrator (no hints, only observation)
- 7 argument categories with first-use bonuses
- Permadeath system with rejection graphics
- MetaMudra Identity Card generation
- Safety protocol with mandatory consent gate

**Repository**
- UNIVERSE.md: Complete Memu arkship lore
- SAFETY.md: Player wellbeing framework
- SETUP_GUIDE.md: Installation and play instructions
- CONTRIBUTING.md: Contribution guidelines
- CODE_OF_CONDUCT.md: Community standards
- LICENSE: Apache 2.0
- Asset documentation (symbols, typography)

**Design Features**
- Compact HUD with decreasing width
- Fraktur typography for atmospheric details
- Small caps for AXIOM voice
- Monospace for Narrator voice
- No emoji policy enforced throughout
