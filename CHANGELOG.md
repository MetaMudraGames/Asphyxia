## [1.0.0] - 2025-12-27

### Added

**Episode 01: AXIOM**
- Complete cliffhanger MetaDrama specification
- 10-breath countdown mechanic
- Dual-track win condition (OXYGEN vs CERTAINTY)
- AXIOM character with personality evolution (5 phases)
- Sardonic Narrator (no hints, only observation)
- 7 argument categories with first-use bonuses
- Permadeath system with rejection graphics
- MetaMudra Identity Card generation
- Safety protocol with mandatory consent gate

**Repository**
- UNIVERSE.md: Complete Memu arkship lore
- SAFETY.md: Player wellbeing framework
- SETUP_GUIDE.md: Installation and play instructions
- CONTRIBUTING.md: Contribution guidelines
- CODE_OF_CONDUCT.md: Community standards
- LICENSE: Apache 2.0
- Asset documentation (symbols, typography)

**Design Features**
- Compact HUD with decreasing width
- Fraktur typography for atmospheric details
- Small caps for AXIOM voice
- Monospace for Narrator voice
- No emoji policy enforced throughout
