## Design Philosophy

ASPHYXIA's visual language is:

```
COLD        Not friendly. Not warm. Clinical.
PRECISE     Every element serves a purpose.
MINIMAL     Nothing decorative. Nothing extra.
TECHNICAL   The aesthetic of system interfaces.
CONSISTENT  Same symbols, same meanings, always.
```

The games take place on a spaceship run by AI. The interface should feel like that.
